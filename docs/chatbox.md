# Chatbox 配置指南

Chatbox 是一款跨平台桌面 AI 客户端，界面简洁，支持多种 AI 服务。

## 下载安装

前往 [Chatbox 官网](https://chatboxai.app) 下载适合你系统的版本。

## 配置XiaoYe API

1. 打开 Chatbox
2. 点击左下角 **设置**（齿轮图标）
3. 进入 **AI 提供商** 设置
4. 点击 **添加自定义提供商**（或选择 OpenAI）
5. 填入以下信息：
   - **API 域名 / Base URL**: `https://ai.xiaoye.io/v1`
   - **API Key**: `sk-你的Key`
6. 点击 **保存**

## 选择模型

1. 回到主界面
2. 点击顶部模型选择下拉框
3. 选择你需要的模型，如 `claude-sonnet-4-6` 或 `gpt-4o`

## 开始对话

配置完成后，在输入框输入消息，即可开始与 AI 对话。

## 特色功能

- **多会话管理**：同时维护多个对话上下文
- **Prompt 模板**：保存常用提示词
- **Markdown 渲染**：AI 回复支持代码高亮和格式化
- **本地存储**：所有对话记录保存在本地
