# Cherry Studio 配置指南

Cherry Studio 是一款功能丰富的桌面 AI 客户端，支持多种 AI 服务提供商。

## 下载安装

前往 [Cherry Studio 官网](https://cherry-ai.com) 下载并安装。

## 配置XiaoYe API

1. 打开 Cherry Studio
2. 进入 **设置** → **模型服务商**
3. 点击 **添加服务商**
4. 选择类型为 `OpenAI 兼容`
5. 填入以下信息：
   - **名称**: `XiaoYe API`
   - **API 地址**: `https://ai.xiaoye.io/v1`
   - **API Key**: `sk-你的Key`
6. 点击 **保存**

## 添加模型

1. 在服务商设置中点击 **获取模型列表**
2. 选择你需要的模型（如 `claude-sonnet-4-6`、`gpt-4o` 等）
3. 开始对话！
