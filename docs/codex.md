# CodeX 配置指南

CodeX 是一款简洁易用的 VS Code AI 编程插件，适合初次接触 AI 编程的用户。

## 安装插件

1. 打开 VS Code
2. 按 `Ctrl+Shift+X` 打开拓展市场
3. 搜索 `CodeX`
4. 点击安装

## 配置XiaoYe API

1. 打开 VS Code 设置（`Ctrl+,`）
2. 搜索 `CodeX`
3. 填入以下配置：
   - **API Endpoint**: `https://ai.xiaoye.io/v1`
   - **API Key**: `sk-你的Key`
   - **Model**: `claude-sonnet-4-6`

## 开始使用

- 选中代码后按 `Ctrl+Shift+A` 唤起 AI 功能
- 在命令面板（`Ctrl+Shift+P`）搜索 `CodeX` 查看所有命令

## 常用功能

| 功能 | 快捷键 |
|------|--------|
| 解释代码 | `Ctrl+Shift+E` |
| 优化代码 | `Ctrl+Shift+O` |
| 生成注释 | `Ctrl+Shift+D` |
| 修复错误 | `Ctrl+Shift+F` |
